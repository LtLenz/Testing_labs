Тест-кейс № 1.Попытка добавить в корзину товар, не выбрав комплектацию 
1. Зайти на сайт https://ru.aliexpress.com/.
2. Перейти в раздел "Телефоны и аксессуары".
3. Выбирать любой товар (прим. Xiaomi Redmi Note 8).
4. Проигнорировать выбор комплектации и цвета.
5. Нажать на кнопку "Добавить в корзину".
Ожидаемый результат: Транзакция не произойдет, появится всплывающее окно с текстом "Пожалуйста, выберите комплектациюЭ

Тест-кейс № 2. Выбор двух телефонов с двумя разными комплектациями в одном заказе.
1. Зайти на сайт https://ru.aliexpress.com/.
2. Перейти в раздел "Телефоны и аксессуары".
3. Выбирать любой товар (прим. Xiaomi Redmi Note 8).
4. Выбрать комплектацию и цвет.
5. Нажать на кнопку "+".
6. Выбрать комплектацию и цвет для нового телефона.
7. Нажать "Добавить в корзину".
8. Нажать на кнопку "Корзина".
Ожидаемый результат: В разделе "Корзина" находится два разных телефона.
Полученный результат: В разделе "Корзина" находится два одинаковых телефона.

Тест-кейс № 3. Ввод несуществующего промо-кода
1. Зайти на сайт https://ru.aliexpress.com/.
2. Перейти в раздел "Телефоны и аксессуары".
3. Выбирать любой товар (прим. Xiaomi Redmi Note 8).
4. Выбрать комплектацию и цвет.
5. Нажать на кнопку "Купить сейчас".
6. В поле промо-код ввести несуществующий промо-код.
7. Нажать кнопку "Применить".
Ожидаемый результат: Промо-код не применится, появится всплывающее окно с текстом "Проверьте правильность введенного промо-кода".

Тест-кейс № 4. Удаление текста из мандаторного поля "Имя". 
1. Зайти на сайт https://ru.aliexpress.com/.
2. Нажать на кнопку "Мой профиль". 
3. Нажать на кнопку "Настройки профиля". 
4. Нажать на кнопку "Изменить профиль".
5. Нажать на кнопку "Редактировать".
6. Удалить текст из полей "Имя".
7. Нажать на кнопку "Представить".
Ожидаемый результат: Провал валидации, появились сообщения "Введите Ваше имя" и "Введите Вашу фамилию".

Тест-кейс № 6. Сменить пароль на пустой. 
1. Зайти на сайт https://ru.aliexpress.com/.
2. Нажать на кнопку "Мой профиль". 
3. Нажать на кнопку "Настройки профиля". 
4. Нажать на кнопку "Сменить пароль".
5. Пройти проверку подлинности e-mail
6. Ввести пароль, не содержащий 6-20 символов и не содержащий спец.символов одновременно.
7. Нажать на кнопку "Запросить".
Ожидаемый результат: Напротив поля ввода появилось сообщение "Параметры не соответствуют требованиям, пожалуйста, введите настройки заново", транзакция не происходит.

Тест-кейс № 7. Сменить пароль на не подходящий по политике безопасности сайта.

Тест-кейс № 8. Отправка пустого письма продавцу
1. Зайти на сайт https://ru.aliexpress.com/.
2. Нажать на кнопку "Мои заказы".
3. Выбрать заказ, который уже находится в статусе "В пути"
4. Нажать на кнопку "Сообщение продавцу". 
5. Не вводить сообщение в поле текстового ввода.
6. Нажать кнопку "Отправить". 
Ожидаемый результат: Ничего не происходит, сообщение не отправляется. 

Тест-кейс № 9. Попытка создания магазина без выбора страны налогообложения.
1. Зайти на сайт https://ru.aliexpress.com/.
2. Нажать на кнопку "Продавцам".
3. Выбрать пункт "Регистрация".
4. Нажать кнопку "Начать". 
5. Оставить поле "Страна налогообложения магазина" пустым. 
6. Нажать кнопку "Продолжить".
Ожидаемый результат: Провал валидации, появилось сообщение "Пожалуйста, выберите страну".

Тест-кейс № 10. Попытка создания магазина без ввода информации о нём.
1. Зайти на сайт https://ru.aliexpress.com/.
2. Нажать на кнопку "Продавцам".
3. Выбрать пункт "Регистрация".
4. Нажать кнопку "Начать". 
5. Выбрать страну налогообложения из списка. 
6. Нажать кнопку "Прожолжить".
7. Оставить каждое из полей пустым.
Ожидаемый результат: Провал валидации, под каждым пустым полем появилось сообщение об ошибке с просьбой ввести запрошенные данные.
